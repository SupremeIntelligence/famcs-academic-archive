# FAMCS Functional Analysis & Integral Equations

## English Version

**Course:** Functional Analysis & Integral Equations  

The course focuses on theoretical foundations and numerical methods for solving nonlinear equations, linear systems, integral equations, and function approximation problems.

---

## Laboratory Works

### Lab 1 — Iterative Methods

- Fixed-point method for nonlinear equations  
- Iterative solution of linear systems  
- Successive approximations for Volterra integral equations  

Main topics:
- Convergence conditions  
- A priori error estimates  
- Numerical implementation of iterative schemes  

---

### Lab 2 — Polynomial and Fourier Approximation

- Approximation using Legendre polynomials  
- Least squares polynomial approximation  
- Trigonometric approximation (Fourier series)  

Main topics:
- Orthogonal systems  
- $( L^2 )$ approximation  
- Error estimation  

---

### Lab 3 — Orthogonal Approximation in Different Bases

- Legendre polynomial expansion  
- Chebyshev polynomial approximation (first kind)  
- Trigonometric polynomial approximation  

Main topics:
- Computation of orthogonal expansion coefficients  
- Weighted inner products  
- Selection of approximation degree by accuracy

## Лабораторные работы

### Лабораторная 1 — Итерационные методы

- Метод простой итерации для нелинейных уравнений  
- Итерационные методы решения систем линейных уравнений  
- Метод последовательных приближений для уравнений Вольтерра  

Основные темы:
- Условия сходимости  
- Априорные оценки погрешности  
- Численная реализация итерационных схем  

---

### Лабораторная 2 — Полиномиальная и тригонометрическая аппроксимация

- Разложение по многочленам Лежандра  
- Аппроксимация методом наименьших квадратов  
- Тригонометрическая аппроксимация (ряд Фурье)  

Основные темы:
- Ортогональные системы  
- Аппроксимация в норме $( L^2 )$
- Оценка погрешности  

---

### Лабораторная 3 — Ортогональная аппроксимация в различных базисах

- Разложение по многочленам Лежандра  
- Аппроксимация многочленами Чебышева I рода  
- Тригонометрический многочлен Фурье  

Основные темы:
- Вычисление коэффициентов ортогональных разложений  
- Взвешенные скалярные произведения  
- Подбор степени аппроксимации по заданной точности
