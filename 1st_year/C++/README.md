# C++ Programming

## English Version

## Course Description

The **C++ Programming** course covers fundamental concepts of programming in C++, including working with arrays, strings, matrices, functions, classes, and basic algorithms. The course emphasizes practical implementation and problem-solving using object-oriented and procedural approaches.

---

## Topics Covered

- Arithmetic operations and string manipulation  
- Series and computational accuracy  
- Arrays and matrices  
- Sorting and searching  
- String processing  
- Data structures (lists, pointers)  
- Classes and object-oriented programming  
- Inheritance and polymorphism  
- Templates and generic programming  
- File handling and binary data  
- Regular expressions  
- STL containers (`list`, `map`, `multimap`) and algorithms  
- Iterators and their usage 

---

## Laboratory Works

- **Lab 1** – Basic arithmetic and string tasks  
- **Lab 2** – Series expansion using Taylor series for `1/(x^3)`  
- **Lab 3** – Arrays: move all prime numbers to the right in an array  
- **Lab 4** – Sorting: Quick sort and counting sort for `float` arrays  
- **Lab 5** – Matrix operations: rotate `MxN` matrix 90° clockwise  
- **Lab 6** – String processing: unique lexemes, numbers, insertion sort, duplicates, file output  
- **Lab 7** – Lists and pointers: filter primes to the end of the list  
- **Lab 8** – Classes: `Matrix` class with rotation and diagonal reflection  
- **Lab 9 (OOP_Lab 3)** – OOP: `Vehicle`, `Car`, `Truck` classes; inheritance, virtual methods, modifying object fields, instance counting
- **Lab 10 (OOP_Lab 4)** – Template classes `MASSIV`, `Students`, `Activists`; binary file I/O, intersection of students and activists, sorting and filtering by activity field
- **Lab 11 (REGEX_Lab5)** – Validating MAC addresses from a file using regular expressions, counting lines, and outputting results to console and file
- **Lab 12 (STL_Lab1)** – Working with a `std::list`: viewing, adding, and removing elements, resizing, and using `transform` and `remove_if` algorithms
- **Lab 13 (STL_Lab2)** – Working with `map` and `multimap`: adding, removing, searching elements, copying data, and modifying values by key

# Программирование на C++

## Русская версия

## Описание дисциплины

Дисциплина **«Программирование на C++»** охватывает основные принципы программирования на C++, включая работу с массивами, строками, матрицами, функциями, классами и базовыми алгоритмами. Основной упор сделан на практическое применение и решение задач с использованием объектно-ориентированного и процедурного подходов.

---

## Изучаемые темы

- Арифметика и работа со строками  
- Ряды и точность вычислений  
- Массивы и матрицы  
- Сортировка и поиск  
- Обработка строк  
- Структуры данных (списки, указатели)  
- Классы и объектно-ориентированное программирование  
- Наследование и полиморфизм  
- Шаблоны и обобщённое программирование  
- Работа с файлами и бинарными данными  
- Регулярные выражения  
- Контейнеры STL (`list`, `map`, `multimap`) и алгоритмы
- Итераторы и их использование  

---

## Лабораторные работы

- **Лабораторная 1** – Основные арифметические и строковые задачи  
- **Лабораторная 2** – Ряд Тейлора для `1/(x^3)`  
- **Лабораторная 3** – Массивы: все простые числа вправо  
- **Лабораторная 4** – Сортировка: Хоара и подсчетом для массивов `float`  
- **Лабораторная 5** – Работа с матрицами: поворот на 90° по часовой стрелке  
- **Лабораторная 6** – Обработка строк: уникальные лексемы, числа, сортировка вставками, дубликаты, вывод в файл  
- **Лабораторная 7** – Списки и указатели: переместить простые числа в конец списка  
- **Лабораторная 8** – Классы: `Matrix` с поворотом и отражением по диагонали 
- **Лабораторная 9 (OOP_Lab3)** – OOP: классы `Vehicle`, `Car`, `Truck`; наследование, виртуальные методы, изменение полей объектов, подсчет экземпляров
- **Лабораторная 10 (OOP_Lab4)** – Работа с шаблонными классами `MASSIV`, `Students`, `Activists`; чтение/запись бинарных файлов, поиск пересечения студентов и активистов, сортировка и фильтрация по полю деятельности
- **Лабораторная 11 (REGEX_Lab5)** – Проверка корректности MAC-адресов в файле с помощью регулярных выражений, подсчет строк и вывод результатов в консоль и файл
- **Лабораторная 12 (STL_Lab1)** – Работа со списком (`std::list`): просмотр, добавление и удаление элементов, изменение размера, применение алгоритмов `transform` и `remove_if`
- **Лабораторная 13 (STL_Lab2)** – Работа с `map` и `multimap`: добавление, удаление, поиск элементов, копирование данных и изменение значений по ключу
