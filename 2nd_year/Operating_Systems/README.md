# Operating Systems

## English Version

## Course Description

The **Operating Systems** course covers fundamental concepts of modern operating systems, including process management, multithreading, synchronization mechanisms, interprocess communication, and file system interaction.

The course focuses on practical implementation of OS-level concepts using system APIs and low-level programming techniques.

---

## Topics Covered

- Process creation and management
- Thread creation and synchronization
- Interprocess communication (IPC)
- Named pipes
- File access synchronization
- Working with binary and text files
- Windows API for process control

---

## Laboratory Works

- **Lab 1** – Process Creation (Main, Creator, Reporter utilities)
- **Lab 2** – Multithreading (min/max and average threads)
- **Lab 3** – Thread Synchronization (main and multiple marker threads)
- **Lab 4** – Interprocess Communication via Shared Binary File (Receiver and Sender processes, FIFO circular queue)
- **Lab 5** – Named Pipes and File Access Synchronization

---

## Technologies Used

- C++
- Windows API
- Console applications
- Binary file processing
- Synchronization primitives

# Операционные системы

## Русская версия

## Описание дисциплины

Дисциплина **«Операционные системы»** охватывает фундаментальные принципы работы современных операционных систем, включая управление процессами, многопоточность, механизмы синхронизации, межпроцессное взаимодействие и работу с файловой системой.

Основной упор сделан на практическую реализацию системных механизмов с использованием API операционной системы и низкоуровневого программирования.

---

## Изучаемые темы

- Создание и управление процессами
- Создание потоков и их синхронизация
- Межпроцессное взаимодействие (IPC)
- Именованные каналы
- Синхронизация доступа к файлам
- Работа с бинарными и текстовыми файлами
- Использование Windows API

---

## Лабораторные работы

- **Лабораторная 1** – Создание процессов (Main, Creator, Reporter)
- **Лабораторная 2** – Многопоточность (поиск min/max и среднего значения)
- **Лабораторная 3** – Синхронизация потоков (main и несколько потоков marker)
- **Лабораторная 4** – Межпроцессное взаимодействие через общий бинарный файл (Receiver и Sender, кольцевая очередь FIFO)
- **Лабораторная 5** – Именованные каналы и синхронизация доступа к файлу
