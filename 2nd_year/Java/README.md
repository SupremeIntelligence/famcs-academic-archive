# FAMCS Java Course

## English Version

**Course:** Java Programming  

**Objective:**  
Learn core concepts of Java programming, object-oriented design, standard libraries, and practical application development.  

---

## Course Content

The course includes theoretical material and practical labs covering:

1. **Basic Java syntax and data types**
   - Primitive types, operators, control structures
   - Input/output operations

2. **Object-Oriented Programming**
   - Classes, objects, inheritance, polymorphism
   - Abstract classes, interfaces

3. **Collections and Generics**
   - Lists, Maps, Sets
   - Iterators and generics usage

4. **Exception Handling**
   - `try-catch` blocks
   - Custom exceptions

5. **File and Stream I/O**
   - Reading/writing text and binary files
   - Serialization

6. **Big Numbers and Math Libraries**
   - `BigInteger`, `BigDecimal`
   - Math functions and approximations (e.g., Taylor series)

7. **Unit Testing**
   - JUnit framework
   - Writing and running tests

8. **String and Date Manipulation**
   - Tokenization and parsing
   - Date formatting and validation

9. **Localization and Formatting**
   - Locale-specific currency and percentage formatting

10. **Project Development**
    - Combining concepts into complete console applications
    - Modular design and utility classes
    - Featured Project: [CoffeeMaker Manager](https://github.com/SupremeIntelligence/FAMCS_Project_IndustrialProgramming?tab=readme-ov-file#famcs-industrialprogramming-project-coffeemaker-manager)￼ – a full Java application demonstrating object-oriented design, file management (TXT, XML, JSON, YAML), encryption, archiving (ZIP/JAR), and automated testing with JUnit. This project showcases applied skills in a real-world scenario.

---

## Lab Structure

Each lab includes:

- **Task Description:** Goal and requirements
- **Input/Output Specification:** Console or file
- **Implementation:** Java classes and methods
- **Testing:** Unit tests (JUnit) and sample runs
- **Documentation:** Comments and README

## Содержание курса

Курс включает теоретические материалы и практические лабораторные работы по темам:

1. **Синтаксис Java и базовые типы данных**
   - Примитивные типы, операторы, конструкции управления
   - Ввод/вывод данных

2. **Объектно-ориентированное программирование**
   - Классы, объекты, наследование, полиморфизм
   - Абстрактные классы, интерфейсы

3. **Коллекции и Generics**
   - List, Map, Set
   - Итераторы и использование generics

4. **Обработка исключений**
   - Блоки `try-catch`
   - Пользовательские исключения

5. **Работа с файлами и потоками**
   - Чтение/запись текстовых и бинарных файлов
   - Сериализация

6. **Большие числа и математические функции**
   - `BigInteger`, `BigDecimal`
   - Математические функции и приближения (например, ряды Тейлора)

7. **Юнит-тестирование**
   - Фреймворк JUnit
   - Написание и запуск тестов

8. **Работа со строками и датами**
   - Разбор строк и токенизация
   - Форматирование и проверка дат

9. **Локализация и форматирование**
   - Форматирование чисел в валюту и проценты с учетом локали

10. **Проектная работа**
    - Сборка консольных приложений с использованием изученных концепций
    - Модульная организация и утилитарные классы
    - Пример проекта: [CoffeeMaker Manager](https://github.com/SupremeIntelligence/FAMCS_Project_IndustrialProgramming?tab=readme-ov-file#famcs-industrialprogramming-project-coffeemaker-manager)￼ – полноценное Java-приложение с обработкой файлов (TXT, XML, JSON, YAML), шифрованием, архивированием (ZIP/JAR) и автоматизированным тестированием с JUnit. Демонстрирует практическое применение изученных навыков.
